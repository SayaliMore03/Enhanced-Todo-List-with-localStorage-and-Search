# Enhanced Todo List (L1 Assignment)

This is an enhanced Todo List application built using HTML, CSS, and JavaScript.  
It supports adding, deleting, completing, searching, and saving tasks using localStorage.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Search tasks in real time
- Persistent storage using localStorage
- Each task stored as an object:
  ```json
  {
    "id": 123456789,
    "text": "Task text",
    "completed": false
  }
